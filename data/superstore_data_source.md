# Dataset Source

This project uses the **Superstore Dataset** obtained from Kaggle.

## Source
- Platform: Kaggle
- Dataset Name: Superstore Dataset Final
- Link: [https://www.kaggle.com/datasets/vivek468/superstore-dataset-final]

## Description
The dataset contains sales and transactional data of a retail superstore. 
It is widely used for data analysis, visualization, and business intelligence projects.

The dataset includes approximately 10,000 records covering orders from **2014 to 2017**, 
providing insights into sales, profit, customer segments, and product categories.

## Features
The dataset contains multiple columns, including:
- Order details (Order ID, Order Date, Ship Date)
- Customer information (Customer ID, Customer Name, Segment)
- Location data (City, State, Postal Code, Region, Country)
- Product details (Category, Sub-Category, Product Name, Product ID)
- Business metrics (Sales, Quantity, Discount, Profit)

## Purpose of Use
This dataset is used in the project to:
- Perform Exploratory Data Analysis (EDA)
- Analyze sales and profit trends
- Identify top-performing categories and regions
- Build visualizations and dashboards
- Derive business insights and recommendations

## Notes
- The dataset is publicly available on Kaggle
- It is commonly used for learning and portfolio projects in data analysis
- No sensitive or personal data is included.


## Project Relevance
This dataset simulates real-world retail business scenarios and helps in understanding how data-driven decisions can improve profitability and operations.
